KeenKeeper

KeenKeeper is a responsive web app to track friendships. Users can see friend details, check contact status, and log interactions like call, text, or video.

Features:
- Responsive design (mobile, tablet, desktop)
- Home page with friend list
- Friend details page
- Quick check-in (Call, Text, Video)
- Timeline page for interaction history
- Stats page with chart
- 404 page

Technologies:
- React.js
- Vite
- React Router DOM
- Tailwind CSS
- Recharts
- React Toastify

Setup:
1. npm install
2. npm run dev
3. npm run build

Notes:
- Friend data is in src/data/friends.json
- Router is created using createBrowserRouter
